## App Details (Max 500 characters)

Price Matrix makes custom dimension pricing simple. Create spreadsheet-style price grids for width × height breakpoints, add option groups with percentage or fixed price modifiers, and assign them to your products. Perfect for art prints, fabric, glass, or any made-to-order business.

Your customers get real-time pricing as they select dimensions and options. When ready to buy, Price Matrix creates a Shopify Draft Order with accurate custom pricing—no manual quotes needed.

**Character count:** 492/500

## Key Features

- **Spreadsheet-like price grid editor** - Define width and height breakpoints with inline editing and keyboard navigation
- **Option groups with price modifiers** - Add flexible choices with percentage or fixed price adjustments (e.g., "Matte Finish +$10" or "Rush Order +15%")
- **Flexible breakpoint system** - Set up any grid size you need (3×3, 5×5, 10×10, etc.) with custom breakpoint values
- **Product assignment** - Assign pricing matrices and option groups to any product in your Shopify catalog
- **Draft Order integration** - Automatic checkout with accurate custom pricing including all selected options
- **REST API** - Fetch pricing and create checkouts from your headless storefront
- **Drop-in React widget** - Pre-built dimension input and option selection component for easy integration
- **CSV import** - Bulk upload pricing grids from spreadsheets (paid plan)
